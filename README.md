# 🟢 TERMINAL DATABASE SYSTEM (v1.0.4)

![Status](https://img.shields.io/badge/STATUS-OPERATIONAL-00ff41?style=for-the-badge&logo=appveyor)
![License](https://img.shields.io/badge/ACCESS-ENCRYPTED-red?style=for-the-badge)

Sistem manajemen database statis berbasis web dengan antarmuka **Terminal Hacker**. Memungkinkan pengguna untuk melakukan input data dan pencarian secara real-time yang terhubung langsung dengan **Google Sheets Engine**.

## 🛠️ CORE ARCHITECTURE
* **Frontend:** HTML5, CSS3 (Matrix Scanline Effect), Vanilla JavaScript.
* **Backend:** Google Apps Script (GAS).
* **Database:** Google Sheets API.
* **Hosting:** GitHub Pages.

## ⚡ FEATURES
- [x] **Secure Entry:** Validasi input otomatis (Execute button terkunci jika data kosong).
- [x] **Quick Scan:** Pencarian data target berdasarkan nama dengan response cepat.
- [x] **Clipboard Sync:** Fitur sekali klik untuk menyalin Nama, Alamat, atau Tanggal Lahir secara terpisah.
- [x] **Matrix UI:** Tampilan retro-future dengan efek scanline dan font monospace.

## 🖥️ SYSTEM INTERFACE
Sistem ini menggunakan skema warna **Neon Green (#00ff41)** dengan latar belakang **Dark Onyx (#0d0208)** untuk menjaga visibilitas tinggi di lingkungan minim cahaya.

## 🚀 DEPLOYMENT NOTES
1. Pastikan **Google Apps Script** telah di-deploy sebagai `Web App`.
2. Izin akses script harus disetel ke `Anyone` agar database bisa dijangkau oleh frontend GitHub.
3. Gunakan format `Plain Text` pada kolom Google Sheets untuk menjaga integritas data tanggal lahir.
